# Эффекты и усложнения по очкам (Neon Pulse)

Ориентировочно: чем выше счёт, тем больше визуального шума и тем короче окно тайминга. Точные значения смотри в `lib/src/game/difficulty.dart` и `lib/src/screens/game_screen.dart`.

## Визуал и геймплей

| Очки (примерно) | Что меняется |
|-----------------|----------------|
| **0–34** | «Разогрев»: только статичные **кольца тайминга** (28 / 48 / 72 / 110 px) и сужающееся **игровое** кольцо; спираль слабее. Нет режима «обод + ракушки» — тап по дистанции до центра как в `_judge`. |
| **≥ 35** | Включается **ring aim**: подсказка обода + зоны GRAZE / RIM / EDGE, спираль **дополнительно ослаблена** (множитель &lt; 1), чтобы не забивать основное кольцо. Игровое кольцо подсвечено **магентовой** сигнатурой, отличной от статичных неоновых колец. |
| **≥ 35** | Тот же порог совпадает с **первым milestone** сложности: плавно уменьшается время цикла сжатия (`shrinkSeconds`). |
| **≥ 70** | **Дрейф центра**: кольца и тап-счёт уезжают по лёгкой фигуре Лиссажу (амплитуда ~10 px; с **≥ 400** — ~18 px). |
| **40 / 120 / 250 / 400** | Ступени **интенсивности спирали** (фон): выше счёт — сильнее движение/насыщенность, но верх **ограничен** и после 35 умножается на коэффициент вместе с ring aim. |
| **50 / 150 / 400** | Бейдж «рост в рейтинге» в UI (косметика). |
| **Milestone 90 → …** | Дальше по таблице `difficulty`: время сжатия продолжает снижаться, **стадия** растёт. |
| **Стадия ≥ 5** *или* **очки ≥ 340** | **Pulse distract**: лёгкое «дёрганье» радиуса основного кольца (синус от пульса), усложняет чтение тайминга. |
| **Стадии 3+** | Потолок **комбо** ↑ (больше максимальный множитель). |

## Справочно (не только от очков)

| Эффект | Условие |
|--------|---------|
| Частицы / всплывающие очки | После каждого попадания/промаха |
| Красная вспышка экрана | Промах |
| Slow-mo цикла | После ULTRA (временное замедление окна) |
| Случайный **×2** к очкам | Случайный бонус после хитов (зависит от счёта только через шанс) |

## Статичные кольца (тайминг по радиусу)

| Радиус от центра | Тир judgement |
|------------------|---------------|
| **&lt; 28** | ULTRA |
| **&lt; 48** | PERFECT |
| **&lt; 72** | GOOD |
| **&lt; 110** | OK |
| **иначе** | MISS (в базовом режиме до 35; с ring aim — ещё проверка по «ободу» и ракушкам) |
